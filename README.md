# Express CRUD App

A simple CRUD (Create, Read, Update, Delete) API built with **Express.js**.  
This app manages users with fields: `name`, `dob`, and `aadhaar`.

---

## Features
- Create a new user
- Get all users
- Get user by ID
- Update user by ID
- Delete user by ID

---

## Installation & Setup
1. Clone this repository
2. npm install
3. npm start
4. Server will start at: http://localhost:3000
